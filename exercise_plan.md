# Exercise Screen Improvement Plan

This document outlines the planned improvements for the Health Exercise tracker.

## 1. Visual Overhaul
- **Dashboard Header**: Implement a glassmorphism dashboard with today's calorie burn estimate and weekly trends.
- **Categorized Grid**: Replace the horizontal routine list with a categorized grid (Cardio, Strength, etc.).
- **Premium Themes**: Use smooth gradients (Orange-to-Sunset) and soft elevations.

## 2. Feature Enhancements
- **Real-time Tracking**: Enable "Starting" a workout with a live timer synced to `FocusBlock`.
- **Calorie Estimation**: Display estimated calories burned per session based on intensity.
- **Grouped History**: Group activity history by date (Today, Yesterday, etc.) for better readability.

## 3. Integration
- **Zen Sync**: Connect mindfulness-related activities (Yoga, Breathing) to the Zen Global Score.
- **Analytics**: Embed core analytics directly into the main exercise screen.
