# 📊 ICE Shield - Point & Level System

## Level Requirements
| Level | Points Required |
|-------|----------------|
| LV 0  | 0 - 99 points  |
| LV 1  | 100 points     |
| LV 2  | 100+200 points     |
| LV 3  | 100+200+300 points     |
| LV 4  | 100+200+300+400 points     |
| LV 5  | 100+200+300+400+500 points     |
| ...   | LV × 100       |

## How to Earn Points

### 🏃 Health - Steps
- **100 steps = 1 point**

### 🍎 Health - Nutrition  
- **A day with < 1500 kcals = +15 points**

### ❤️ Social - Affection
- **Each contact = +30 points**
- **Every 5 affection = +10 points**

### 💰 Finance - Wealth
- **Every $1000 in savings = +50 points**
- **Every 5% investment return = +10 points**

### 💼 Projects - TBD

task 2 points
project 50 points

## Applies to All Categories
- Health (steps, calories)
- Social (contacts, affection)
